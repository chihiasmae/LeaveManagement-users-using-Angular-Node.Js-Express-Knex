### Technologies
Project is created with:
* Angular CLI: 8.1.1  & Typescript
* Node: 10.15.3
* Bootstrap: 3.4.1 & CSS
* Angular material: 8.1.1
* knex: 0.19.2

## Setup
To run this project, install it locally using npm:

* run backend:
$ cd yourProject
$ npm install
$ cd backend
$ node server.js

* in new terminal run frontend:

$ cd yourProject
$ ng serve -o
